Wav.Net
=======

Is a fully featured .Net library<sup>ยง</sup> for transcoding (reading, writing & creating) `.wav` (wave) files.

Current support:

 - Bit depth: 8 to 64-bits,
 - Sample rate: 1 to 4,294,967,296Hz,
 - Channels: 19,
 - Audio format: PCM and IEEE floating-point.
 
---

 <sup>ยง *Requires .Net 2.0 or higher.*</sup>
